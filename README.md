This project focuses on predicting the Fire Weather Index (FWI) using machine learning techniques applied to the Algerian Forest Fire dataset. 
FWI is a standard indicator used to measure the potential risk and intensity of forest fires based on weather conditions and fuel moisture content. 
The model takes multiple climatic and fire-related parameters such as temperature, humidity, wind speed, rainfall, and fire weather indices to estimate the FWI value. 
Based on the predicted FWI, the system identifies the level of fire danger ranging from low to extreme. 
The application is deployed using Flask and provides an interactive web interface that allows users to input parameters and instantly obtain predictions, helping in early wildfire risk assessment and decision-making
